Comandos para executarmos no GIT após editarmos o código

- Abrir o CMD:

- Ir para a página do projeto pelo prompt de comando:

- C:\projects\html\pagehtml

- Usar os seguintes comandos:

git status carrega enter - comando que permite a verificação do status dos arquivos em seu repositório local

git add . carrega enter - comando que prepara as alterações realizadas no repositório local para serem sincronizadas no servidor remoto da aplicação

git commit -m "escreva sua contribuição" carrega enter - comando que registra as alterações realizadas

git push carrega enter - comando utilizado para enviar as alterações feitas em um repositório local para um repositório remoto
